## 🚀 Quick Start Guide

### **1. Install Dependencies**
```bash
cd backend && npm install
cd ../frontend && npm install
```

### **2. Configure Backend** 
Edit .env:
```env
MONGODB_URI=mongodb://localhost:27017/task_tracking
JWT_SECRET=your_secret_here
EMAIL_USER=your_email@gmail.com (optional)
```

### **3. Start Everything**
**Terminal 1:** `mongod` (MongoDB)
**Terminal 2:** `cd backend && npm start` (Backend on port 5000)
**Terminal 3:** `cd frontend && npm start` (Frontend on port 3000)

### **4. Test Accounts**
- **Admin**: admin@test.com / admin123
- **Employee**: employee@test.com / emp123
